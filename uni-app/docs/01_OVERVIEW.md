# УниВест — Обзор проекта

## Что это такое

**УниВест** — полноценное мобильное приложение для студентов и преподавателей высшего учебного заведения. Приложение решает задачу централизованного информирования: все новости, объявления и учебные материалы — в одном месте, на телефоне, с мгновенными уведомлениями.

Проект является дипломной работой по теме:
> «Разработка мобильного приложения для рассылки новостей и образовательных ресурсов высшего учебного заведения с использованием технологии React Native»

## Что умеет приложение

### Для студентов и преподавателей
- Просматривать новости университета с фильтрацией по категориям
- Читать полный текст новостей с изображениями
- Искать новости по ключевым словам
- Просматривать образовательные ресурсы (ссылки, файлы, материалы)
- Фильтровать ресурсы по категориям
- Открывать внешние ссылки (библиотеки, платформы, файлы)
- Получать push-уведомления о важных событиях
- Просматривать свой профиль

### Для администратора (дополнительно)
- Создавать, редактировать и удалять новости
- Публиковать/снимать с публикации новости
- Управлять образовательными ресурсами
- Управлять категориями (создание, редактирование, удаление)
- Отправлять push-уведомления всем пользователям с шаблонами
