## LoginScreen — Вход в систему

**Файл:** `src/screens/LoginScreen.tsx`

Форма входа с полями Email и Пароль. Кнопки быстрого заполнения тестовых аккаунтов (полезно для демонстрации). При успешном входе Navigation автоматически переключается на нужный набор экранов в зависимости от роли.

**Логика:**
1. Пользователь вводит email + пароль
2. Вызывается `authStore.login(email, password)`
3. Стор вызывает `POST /auth/login`
4. Токен сохраняется в Expo SecureStore
5. Zustand обновляет `user` и `isAuthenticated = true`
6. Navigation определяет роль и показывает нужные вкладки

## AdminNotificationScreen — Рассылка уведомлений

**Файл:** `src/screens/admin/AdminNotificationScreen.tsx`

Информационная карточка. Три шаблона быстрого заполнения. Поля заголовка и текста. Предпросмотр уведомления (как будет выглядеть на телефоне). Кнопка «Отправить всем» с подтверждением.

**Процесс отправки:**
1. Администратор заполняет форму
2. Нажимает «Отправить всем»
3. Подтверждает в диалоге
4. Вызывается `POST /notifications/send`
5. Backend получает все device_tokens из БД
6. Отправляет запрос на Expo Push API
7. Expo доставляет уведомления на устройства пользователей
